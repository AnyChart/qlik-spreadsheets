# Spreadsheets Extension for Qlik Sense

**Bring Excel-style spreadsheets directly into your Qlik apps.**

The **Spreadsheets** extension by **AnyChart** adds a familiar spreadsheet interface inside Qlik Sense. Users can work with formulas and functions, edit and format data, create in-cell sparklines or full charts, and do more of what they know from Microsoft Excel — all while staying connected to Qlik’s associative engine. Existing Excel files and templates can be opened, modified, and saved back as needed.

[<img src="https://www.anychart.com/blog/wp-content/uploads/2025/09/Spreadsheets-for-Qlik-for-GitHub.png" width="75%" alt="Screenshot of AnyChart Spreadsheets Extension for Qlik Sense">](https://qlik.anychart.com/download)

## Key Features
- Excel-like UI
- Formulas, functions, and calculations
- Inline editing and collaboration
- Flexible formatting
- Support for common keyboard shortcuts
- Sparklines and other charts
- Import/export of full Excel files
- Live integration with Qlik data models
